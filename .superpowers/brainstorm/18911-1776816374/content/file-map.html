Section 3: Files to Create & Change

Every file touched — new in green, modified in amber

New files

+ src/app/api/tools/web/route.ts
Receives {message, locale, history}. Calls Gemini with Grounding tool enabled. Returns {text, sources[]}.
+ src/app/api/tools/youtube/route.ts
Receives {message, locale}. Extracts or searches YouTube ID → fetches transcript via YouTube Data API → Gemini summarises. Returns {text, video: {title, channel, duration, thumbnailUrl, videoId}}.
+ src/app/api/tools/image/route.ts
Receives {prompt, locale}. Translates prompt to English if needed → calls Imagen 3 via Vertex AI → returns {imageBase64, mimeType}.
+ src/app/api/tools/transcribe/route.ts
Receives multipart/form-data with audio blob. Encodes to base64 → sends to Gemini as audio part → returns {text}.
+ src/lib/intent.ts
detectIntent(message: string): "web" | "youtube" | "image" | "chat". Pure function, regex + keyword patterns for FR/MG/EN.
+ src/lib/gemini.ts
Shared Gemini fetch helper. callGemini({systemPrompt, contents, tools?, config?}) — eliminates the duplicated raw fetch across all route handlers.
+ src/hooks/useVoiceRecorder.ts
React hook. Manages MediaRecorder lifecycle — idle → recording → transcribing → done. Returns {state, start, stop, transcript}.
+ src/types/message.ts
Extended Message type. Adds sources?, image?, video?, file? fields to the existing interface.
